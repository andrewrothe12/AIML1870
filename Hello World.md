# Retro Vibe Code Webpage

A retro 80s synthwave-inspired webpage featuring an animated grid background and sunset aesthetic.

## Features

- Glowing neon text effects
- Animated purple and black grid background
- Retro sunset gradient
- Responsive design
- Scanline overlay for authentic retro feel

## How to View

Simply open `retro_page.html` in your web browser.

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (CSS animations)
